# Research-Assignment-1
This was a research assignment that covers data fundamentals
This assignment explored data fundamentals and data management concepts, focusing on how organizations store, structure, and analyze large volumes of data. It covered the role of databases in enabling efficient data storage, retrieval, and real-world applications such as healthcare systems.

Key concepts included:

Databases and their importance in managing structured data efficiently
Differences between operational databases and data warehouses, particularly for real-time vs historical data analysis
The concept of a data lake as a flexible storage system for raw, unstructured data
Data marts as smaller, focused subsets of data warehouses for specific business needs
Data modeling (conceptual and logical) to organize and define relationships between data
An introduction to data mining for extracting insights and patterns from large datasets

Overall, the assignment highlighted how different data systems and models support decision-making, scalability, and efficient data analysis in modern organizations.
